# Examples

This directory contains the educational modules of SCSL.

The canonical structure is:

- `examples/contracts/<module>/` for educational Solidity sources and module guides
- `examples/test_examples/<module>/` for attack and fix tests

Each module is organized as a self-contained learning unit:

- a vulnerability guide in English
- a vulnerable contract
- an attacker contract
- a fixed contract
- attack tests
- fix validation tests
